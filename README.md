# منصة التنسيق الرقمي - الجزائر 🇩🇿

منصة لإدارة التنسيق الرقمي بين المديرية والمراكز والمصالح، تتضمن:
- 📋 إدارة الانشغالات والمخاوف
- 💬 نظام محادثات متكامل
- 🎥 قاعة اجتماعات فيديو (WebRTC)
- 📊 لوحة تحكم وإحصائيات
- 🔐 نظام صلاحيات متعدد المستويات
- 🌐 مزامنة فورية عبر Firebase

## 🚀 التشغيل

### الطريقة 1: فتح مباشر (بسيط)
- افتح `index.html` في المتصفح
- ⚠️ بعض ميزات Firebase قد لا تعمل عبر `file://`

### الطريقة 2: خادم محلي (موصى به)
```bash
# Python
python -m http.server 8000

# أو Node.js
npx serve .
```

ثم افتح: `http://localhost:8000`

---

## 🔧 الإعداد

1. أنشئ مشروع على [Firebase Console](https://console.firebase.google.com)
2. فعّل **Realtime Database**
3. انسخ بيانات الإعداد إلى `js/config/firebase.js`:

```javascript
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  databaseURL: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
```

4. ضع قواعد الأمان في Firebase Console:

```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```

## 👥 الأدوار

| الدور | الوصف |
|---|---|
| 🏛️ **المديرية** | صلاحية كاملة - ترسل الانشغالات، تستقبل الردود، تدير الاجتماعات |
